# Protector-de-Chupet-n-Dise-o-Tipo-Cohete-3D
Este proyecto consiste en el diseño de un protector para bombones tipo chupetín, inspirado en la forma de un cohete espacial. El objetivo es ofrecer una solución higiénica, reutilizable y atractiva, especialmente pensada para niños, combinando funcionalidad con un diseño creativo.
